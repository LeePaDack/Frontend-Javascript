변수 (Variable)

        * 메모리의 값을 계속 바꿔가면서 저장할 수 있는 공간
          변하는 수
        * 변수에 이름을 붙이고(변수명) 원하는 값을 넣어주면 
          변수명이 필요한 곳에서 변수명을 호출해서 지정된 값을 사용할 수 있음
          
        
변수명 이름 작명 규칙

        - 의미 있는 이름 작성(a, b, c 이런식의 작성은 X)
        - 대, 소문자 작성
        - 특수문자는 _(언더스코어) 만 사용가능(단, 첫 글자는 제외)
        - 중간에 대문자를 작성해주는 카멜표기법 작성

        [카멜 표기법(Camel Case)]
        문장에서 여러 단어를 변수명으로 작성할 때 이어지는 첫 번째 단어를 대문자로 작성
        member name -> memberName


        특수 문자 중에서 $ 가 존재
        $ 는 혼자 사용 가능
        $숫자 사용 가능
        $변수명 처럼 맨 앞에도 사용 가능
        -> 추후 jQuery 표기 문법으로 사용할 문자 ($는 잘 사용 X, jQuery 에서 쓰기 때문)

        $ 문자는 어떤 표현식을 사용할 때 사용하기도 함

Java Srcipt 변수의 종류와 특징

        var   : 변수, 변수명 중복 O, 덮어쓰기 가능
            -> 변수명 재사용 가능, 변수명 안에 있는 내용 변경 가능
        let   : 변수, 변수명 중복 X
            -> 변수명 재사용 불가능, 변수명 안에 있는 내용 변경 가능 
        const : 상수, 변수명 중복 X (제일 많이 사용)
            -> 모두 불가

javascript 의 자료형

        - 문자열을 String 사용 대신에 " " 표현 후 작성
            const 문자열 = "가나다";
        - 정수를 표현할 때는 int 대신 숫자 작성
            const 정수 = 123;
        - 논리값을 표현할 때는 boolean 대신 true false 작성
            const 논리값 = true;

        _ null -> 참조하는 값이 없음 *주로 오타에서 발생
